Mejora el posicionamiento en buscadores evitando el contenido duplicado en la paginación de comentarios de wordpress 2.7 o superior.

== Description ==

El objetivo de este plugin es mejorar el **posicionamiento de buscadores** colocando las etiquetas meta noindex y nofollow en la **paginación de comentarios** (disponible desde la version WordPress 2.7) evitando de esta manera el duplicado de contenidos, se aplica a todas las paginas individuales del blog. La forma de verificar si tienes contenido duplicado es ingresando a tu cuenta de Google Webmaster y te daras cuenta que las paginaciones en los comentarios aparecen como contenido duplicado.

Se requiere:

* PHP 4.1 (o mayor)  
* WordPress 2.7 o superior

Para mayor información visita:
[Blog de Tecnologia](http://www.wihe.net/)

== Installation ==

- Desempacar el Archivo en un directorio vacio.
- Subir el archivo guardado al directorio "/wp-content/plugins/". 
- Entra a tu panel de administracion o dashboard y activa el plugin.
